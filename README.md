# Shahzaib Nawaz — Portfolio & Blog

A personal portfolio and blog built with the modern Next.js stack.  
Live at: https://portfolio-blog-szn.vercel.app/

---

## About

This is a fast, responsive, and SEO-optimized portfolio website where I showcase my projects and write technical blog posts.

The blog is powered by MDX, and the entire site is designed with performance, accessibility, and clean developer experience in mind. Pages are statically generated wherever possible to ensure speed and scalability.

---

## Tech Stack

- **Next.js (App Router)** — routing, layouts, Server Components
- **TypeScript** — type safety across the entire project
- **Tailwind CSS v4** — utility-first styling with modern CSS-first config
- **shadcn/ui** — accessible and reusable UI components
- **MDX** — writing blog posts with React components inside Markdown
- **next-themes** — dark/light mode support
- **Vercel** — deployment and hosting

---

## Features

- ⚡ Fast, static-first pages using Next.js App Router
- 📝 MDX-powered blog with frontmatter support
- 🖼️ Optimized images using `next/image`
- 🌙 Light / Dark mode with system preference support
- 🔍 SEO optimized (metadata API, Open Graph, sitemap, robots.txt)
- 📱 Fully responsive design across all devices
- 🧩 Reusable UI components using shadcn/ui
